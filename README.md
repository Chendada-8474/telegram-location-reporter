# 魚塭黑面琵鷺及雁鴨回報系統

這是一個用 telegram bot 所製作的黑面琵鷺及雁鴨點位回報系統，讓大家可以方便快速回報點位。

## 指令列表
也可以在對話中輸入 <font color="#007FFF">/help</font> 查看所有指令。

- <font color="#007FFF">/help</font> - 查看所有指令。
- <font color="#007FFF">/signup</font> - 跟管理員申請回報權限
- <font color="#007FFF">/contact</font> - 聯絡管理員，若有問題可以直接利用 telegram 與管理員聯繫。


## 1. 下載 Telegram App

請先從 Google Play 或者是 App Store 下載 Telegram。
下載下來後只要照著設定註冊完即可使用。


## 2. 搜尋帳號

1. 請在最上面的 Search 搜尋 "Cane Toad Reporting System" 。或者是掃描這個 QR code 也可以找到回報系統帳號。

<img src="https://i.imgur.com/WALVqZx.jpg" alt="" width="300"/>


2. 點選 Cane Toad Reporting System 。按下 start 就可以開始囉！

<img src="https://i.imgur.com/PNgD7ts.png" alt="" width="300"/>

<img src="https://i.imgur.com/CVPxoG0.png" alt="" width="300"/>

*目前為測試系統，之後正式開始使用之後會換另一個機器人帳號*

## 3. 跟系統申請回報權限

開始回報之前，需要先申請權限，再由管理者審核加入。

### 申請方式

請直接輸入 "/signup" ，來申請權限，並選擇你服務的單位。申請後就等一下下，管理員有看到就會幫你審核囉！

如果申請過了，系統會通知你。就可以開始回報點位了。



## 4. 黑琵雁鴨點位回報


### ios 系統


1. 點選左下角的迴紋針圖案，打開傳送選項，點選 "Location"。

<img src="https://i.imgur.com/u9XVd7C.png" alt="" width="300"/>

<img src="https://i.imgur.com/zfUfsEY.png" alt="" width="300"/>

2. 傳送點位
    - 請滑動地圖，盡量選擇黑琵或雁鴨所在的魚塭中心點。
    - 請不要選擇 Live Locaion!!
    - 可以切換成衛星地圖更好抓魚塭中心點喔！

<img src="https://i.imgur.com/nVtP8n4.png" alt="" width="300"/>

<img src="https://i.imgur.com/u0F1VMm.jpg" alt="" width="300"/>


3. 選擇物種，緊接著輸入數量。


<img src="https://i.imgur.com/9rlxp0N.png" alt="" width="300"/>

<img src="https://i.imgur.com/f4ZsyGq.png" alt="" width="300"/>



4. 有多個物種可以繼續按物種的按鈕新增物種，輸入錯誤可以在按一次一樣的物種刪除。例如再按一次琵嘴鴨，琵嘴鴨就刪除了。

<img src="https://i.imgur.com/5fSfem5.png" alt="" width="300"/>

<img src="https://i.imgur.com/s9KiaK0.png" alt="" width="300"/>



5. 按下確認就成功上傳囉！

<img src="https://i.imgur.com/UWRioKg.png" alt="" width="300"/>

## 5. 其他提示

### 重新選擇
如果輸入到一半想要重新選擇點位，可以直接再分享一個點位重新選擇，或者是按取消將紀錄清除。

### 請勿事後輸入
系統會自動抓分享點位當下的時間，請勿事後輸入，造成紀錄時間不正確。
